# 变更记录

本文件记录 AI 产品工程框架的重要版本变化。当前运行状态以 [Framework 项目 Context](12_框架项目Context/README.md) 为准。

## [未发布：0.2.0]

目标：将 Context 与 Harness 从框架定义转化为可执行、可验证的工程能力。

### Context 可执行化

- 建立框架级、项目级、阶段级、任务级四级 Context 和独立决策经验回写层；
- 建立事实源与状态、项目 Pack、阶段 Context、任务 Pack、装配、冲突和回写规范；
- 建立 Context 完整性检查清单和五类候选模板；
- 完成 Framework 自应用基线，历史评分 93/100；自应用不替代真实业务验证。

### YouYu 参考工程进度

- 完成工程边界加固、iOS 真机构建安装启动、服务端和采集工程构建，以及既有远程重复构建检查；
- 确认首个正式业务切片为“手机号验证码登录注册与个人资料管理”；
- 完成产品范围、不做清单、业务规则、验收断言、用户流程、信息架构、页面清单和交互说明；
- YouYu 已采用统一数据库基础规范，Framework 登记为候选工程规格，尚待正式业务表运行验证；
- 高保真候选更新为 `v0.1.0-draft.6`，检查结论为 `conditional_pass`，等待维护者逐页批准；
- 主导航已对齐正式 iOS 工程，未建设空入口、重复事实源、已替代图标和临时预览地址固化问题已关闭；
- 登录页覆盖需要确认当前协议版本，以及已确认当前协议版本、无需重复勾选两种候选状态；
- 候选原型检查从完整工程构建中拆分为独立轻量工作流；完整工程检查改为按代码路径触发；
- 候选原型检查覆盖 JavaScript、JSON、版本、精确主导航、协议双状态、唯一事实源、本地静态引用和页面访问；
- 最近候选原型远程运行受 GitHub Actions 账户账单或额度限制影响而未启动，维护者决定暂缓远程成功证据；
- Framework 项目 Context Pack 更新为 `0.2-A.15`，阶段 Context Pack 更新为 `2.4`；
- 新增 TASK-20260715-011 至 TASK-20260715-014，记录产品体验、高保真、远程检查状态和检查拆分。

### 当前开发边界

- YouYu 正式业务实现为 `blocked`，正式业务参考工程验证为 `not_started`；
- 高保真人工批准、业务表、OpenAPI、错误码、示例数据、客户端映射、实现边界和三层验证尚未完成；
- 历史安全、共享会话、下游网络隔离、采集待审链路和 iOS 真机体验验收仍未关闭；
- Context 模板和数据库规范继续为 `candidate`；
- YouYu 项目级候选原型检查不代表 Harness 里程碑 B 已开始；
- 本轮不修改 `VERSION`，稳定版本保持 v0.1.6。
