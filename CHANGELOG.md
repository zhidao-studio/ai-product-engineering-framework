# 变更记录

本文件记录 AI 产品工程框架的重要版本变化。当前运行状态以 [Framework 项目 Context](12_框架项目Context/README.md) 为准。

## [未发布：0.2.0]

目标：将 Context 与 Harness 从框架定义转化为可执行、可验证的工程能力。

### Context 可执行化

- 建立框架级、项目级、阶段级、任务级四级 Context 和独立决策经验回写层；
- 建立事实源与状态、项目 Pack、阶段 Context、任务 Pack、装配、冲突和回写规范；
- 建立 Context 完整性检查清单和五类候选模板；
- 完成 Framework 自应用基线，历史评分 93/100；自应用不替代真实业务验证。

### YouYu 参考工程进度

- YouYu 发布 `v0.1.2` 验证加固补丁；
- 修正 Sa-Token 编译包路径、验证码失败事务、主键更新、共享 Redis 配置、Mapper 扫描和契约漂移；
- OpenAPI 权威版本升级到 v0.1.2，成功码、Token 和资料字段与实现一致；
- 新增安全、请求、资料和字符串身份单元测试；
- 新增数据库结构断言和 Maven/MySQL/Redis/Gateway/App 联合检查工作流；
- 新增 SERVER-CHECK-002，静态结论为 `conditional_pass`；
- YouYu TASK-011 转为 `blocked_external_runtime_evidence`；
- GitHub Actions 仍没有 v0.1.2 运行结果。

### 当前开发边界

- YouYu 服务端状态为 `implemented_not_runtime_validated`；
- 自动检查已定义不等于已经执行通过；
- YouYu 正式业务验证保持 `not_started`；
- Context 模板和数据库规范继续为 `candidate`；
- Harness 里程碑 B 尚未开始；
- Framework 仓库仍为 Public，待维护者切换 Private。

## [0.1.8] - 2026-07-16

版本定位：同步 YouYu v0.1.2 静态缺陷修复和可执行验证资产，并将“实现、检查定义、检查执行”三种状态明确分离。

- 同步编译、事务、配置、契约和主键更新修复；
- 同步单元测试、数据库断言和联合工作流；
- 同步 v0.1.2 OpenAPI 唯一事实源；
- 同步 SERVER-CHECK-002 与 TASK-011 外部运行阻塞；
- Framework Context Pack 升级，稳定版本更新为 v0.1.8；
- 不提升候选资产成熟度，不启动 Harness B。

报告：[v0.1.8 YouYu验证加固同步报告](10_版本演进/v0.1.8YouYu验证加固同步报告.md)。

## [0.1.7] - 2026-07-16

- 稳定版本更新为 v0.1.7。
