# 变更记录

本文件记录 AI 产品工程框架的重要版本变化。

## [未发布：0.2.0]

目标：将 Context 与 Harness 从框架定义转化为可执行、可验证的工程能力。

### Context 可执行化

- 确立“框架级、项目级、阶段级、任务级”四级 Context 作用域与独立决策经验回写层。
- 新增 Context 工程总览图和模块入口。
- 完善 Context 与记忆管理总览，区分权威事实源、Context Pack、模型会话和自动记忆。
- 新增事实源与状态规范，定义元数据、状态、版本、新鲜度、敏感级别和权威索引。
- 新增项目 Context Pack、阶段 Context、任务 Context Pack 规范。
- 新增 Context 装配、预算、冲突、缺失、过期和敏感信息处理规范。
- 新增决策与经验回写规范和 Context 完整性检查清单。
- 提供项目、阶段、任务、冲突和经验回写五类候选模板。
- 统一 Context 机器状态枚举、中文显示和 Pack 版本字段。
- 修正任务 Context 示例的生命周期阶段映射。

### Framework 自应用验证

- 新增 DEC-007，确定“先自应用 Context，再进入业务参考工程”的验证顺序。
- 建立 `12_框架项目Context`，保存 Framework 自身的项目、阶段、任务和检查记录。
- 使用候选模板建立 Framework 项目 Context Pack、里程碑 A 阶段 Context 和任务 Context Pack。
- 完成 CTX-CHECK-001，自应用 Context 基线 P0 全部通过，成熟度评分 93/100。
- 项目维护者已经确认自应用结果。
- 确认自应用只能证明仓库记忆和一致性管理，不能替代真实业务产品验证。

### 当前状态

- Context 规范、检查清单和候选模板已经建立。
- Framework 自应用基线检查已经通过。
- 候选模板成熟度仍为 `candidate`，下一步必须通过真实业务参考工程任务验证。
- 尚未建设自动 Context 装配器、向量知识库、版本门禁或完整 Harness 脚本。

## [0.1.1] - 2026-07-12

版本定位：v0.1.0 的全量 Review 与治理修订，不改变框架宪法、三平面模型、十阶段生命周期和五大基础设施。

### 全量复核与事实源治理

- 完成全仓库 Review，覆盖根文件、宪法、全局模型、角色、Context、Harness、Skills、Agents、Loop、模板、参考工程、Roadmap、设计决策和图片资产。
- 删除 `02_核心模型`、`05_设计决策记录` 等重复事实源，保留 `02_全局模型` 和 `11_设计决策` 作为权威目录。
- 合并重复的适用场景文档，恢复单一事实源。
- 将模拟用户验收与发布交付拆分为独立阶段，统一十阶段产品价值生命周期。
- 修复 ChatGPT 概念总览图片引用断链。
- README 显式列出愿景、场景、原则和边界四份宪法文档。
- 修正 `README结构规范.md` 中已经失效的目录和版本描述。

### 版本治理

- 新增《版本管理规范》。
- 新增 DEC-008，采用语义化发布版本与独立开发里程碑。
- 明确 v0.1.1 为稳定修订版本，v0.2.0 为目标开发版本。
- 明确 `A/B/C` 是 v0.2.0 内部里程碑，不是发布版本。
- 明确 Context Pack 版本、稳定发布、目标发布和来源版本使用独立字段。

### 仓库治理

- 新增 `CONTRIBUTING.md`，定义贡献分类、事实源、验证、版本、图示和敏感信息要求。
- 新增 `v0.1.1全量复核报告.md`。
- README、AGENTS、Roadmap、项目 Context 和设计决策索引同步升级至 v0.1.1 基线。
- 明确仓库尚未选择正式开源许可证，许可证由项目维护者决定。

### 版本边界

- 本版本只修订 v0.1.0 既有框架范围和治理质量。
- Context 详细规范、候选模板和自应用验证属于未发布的 v0.2.0 开发内容。
- 本版本不提供自动 Agent 平台、完整 Skill 库、可运行的全套 Harness 或无人值守发布系统。

## [0.1.0] - 2026-07-12

### 新增

- 建立 AI 产品工程框架的愿景、定位、边界和适用场景。
- 建立“产品价值生命周期、AI 工程基础设施、治理与组织记忆”三平面模型。
- 定义 Context、Harness、Skills、Agents、Loop 五大 AI 工程基础设施。
- 将高保真预览、模拟用户验收和持续反馈纳入正式生命周期。
- 建立人类角色与 AI Agent 角色体系。
- 建立设计决策记录机制，避免关键结论只保留在模型会话中。
- 建立模板资产、参考工程和版本演进入口。
- 增加 Mermaid 图、仓库内 SVG 和 ChatGPT 生成的真实图片资产。

### 版本边界

- 本版本以框架立项和完整地图为目标。
- 暂不提供自动化 Agent 平台、完整 Skill 库或无人值守发布系统。
