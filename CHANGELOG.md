# 变更记录

本文件记录 AI 产品工程框架的重要版本变化。当前运行状态以 [Framework 项目 Context](12_框架项目Context/README.md) 为准。

## [未发布：0.2.0]

目标：将 Context 与 Harness 从框架定义转化为可执行、可验证的工程能力。

### Context 可执行化

- 建立框架级、项目级、阶段级、任务级四级 Context 和独立决策经验回写层；
- 建立事实源与状态、项目 Pack、阶段 Context、任务 Pack、装配、冲突和回写规范；
- 建立 Context 完整性检查清单和五类候选模板；
- 完成 Framework 自应用基线，历史评分 93/100；自应用不替代真实业务验证。

### YouYu 参考工程进度

- 首个正式业务切片确认为“手机号验证码登录注册与个人资料管理”；
- 产品、体验、高保真候选、工程差异和既有引用扫描已经建立；
- 维护者确认 12 项关键业务与安全参数；
- YouYu 发布 `v0.1.1`，新增 App 独立账号域数据库迁移、回滚、正式 OpenAPI、错误码、字符串身份兼容、Bearer 网关鉴权和服务端基础实现；
- App 用户与管理员 `sys_user`、旧 `/api/auth/v1/**` 和旧 `BaseEntity` 保持隔离；
- 真实短信、头像文件和地区数据未接入时失败关闭；
- YouYu 进入 TASK-011 质量与安全验证阶段，数据库、构建、联合运行、iOS 和端到端证据尚未完成；
- 高保真 `v0.1.0-draft.6` 仍待维护者逐页批准；
- GitHub Actions 账户账单或额度限制仍影响远程证据。

### 当前开发边界

- YouYu 服务端基础实现状态为 `implemented_not_validated`，正式业务参考工程验证仍为 `not_started`；
- 版本号、代码提交和静态文档不等于数据库、运行、安全、iOS 或生产验收通过；
- Context 模板和数据库规范继续为 `candidate`；
- Harness 里程碑 B 尚未正式开始；
- Framework 仓库仍为 Public，待维护者切换 Private。
