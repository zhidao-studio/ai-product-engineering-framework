## 4. 静态检查

- [ ] 文件与版本可定位；
- [ ] 路径和 HTTP 方法一致；
- [ ] 字段名、类型、必填和枚举一致；
- [ ] 数据表、索引、触发器和实体一致；
- [ ] 错误码、HTTP 状态和客户端映射一致；
- [ ] 鉴权与身份字段不信任客户端伪造值；
- [ ] 生成代码没有未解释差异。
