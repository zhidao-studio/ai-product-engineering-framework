## 1. 版本定位

v0.1.10 是 Framework 的 Context 提交可追溯性、发布一致性检查和 YouYu v0.1.4 回写修订版本。它不改变 Framework 核心模型，不代表 YouYu 已通过运行验证，也不代表里程碑 A 已退出。

## 2. Framework 已修复

- 将无效 `baseline_commit` 修正为可解析的 Framework 提交；
- TASK-020 区分 Framework `source_commit` 与 YouYu `reference_source_commit`；
