## 1. 版本定位

v0.1.9 是 Framework 的全仓事实源收敛、YouYu v0.1.3 静态修复同步和发布治理修订版本。它不改变框架核心模型，不代表 YouYu 运行验证完成，也不代表 v0.2.0 里程碑 A 已退出。

## 2. YouYu v0.1.3 已修复

### 数据库审计

- 保留 v0.1.1 历史建表迁移；
- 新增 v0.1.3 触发器修订和安全回滚；
- 插入时无条件覆盖创建与修改时间；
- 更新时保护历史创建时间并刷新修改时间；
- 触发器名称统一为 `trg_<table>_before_insert/before_update`；
- 检查触发器正文、主键、关联 ID、审计列和必要唯一索引。

### 验证码并发

- 使用 `SELECT ... FOR UPDATE`；
- 错误次数在事务锁内按主键更新；
- 第五次错误固定为 `LOCKED:5`；
- 同一验证码并发正确请求只能有一个消费成功；
- 协议等后续业务失败回滚验证码占用。

### 配置与契约

- App/Admin 不默认激活 dev；
- 运行环境必须显式提供数据库连接配置；
- H2 仅保留测试范围，不能在部署时静默回退内存库；
- 公共配置移除完整 SQL 参数输出；
- Gateway 下游地址支持环境变量；
- 协议数组最多两项，空对象、重复对象和同类型冲突版本失败；
- 昵称和简介去除首尾空白后按 Unicode 码点计算；
- Trace ID 限制为 32 位小写十六进制；
- 短信适配边界区分发送号码和脱敏审计号码；
- 网络来源频控由 YouYu TASK-012 管理，状态为 `not_implemented`。

### Context 与验证链

- YouYu 项目 Context 采用 Framework Context Pack 字段；
- TASK-011 保持运行证据阻塞；
- 产品规则不再保存动态实现状态；
- 高保真仍待批准，iOS 正式实现未开始；
- OpenAPI v0.1.3 成为当前唯一权威契约；
- 新增本地可执行检查脚本，Actions 调用同一脚本；
- SERVER-CHECK-003 静态复核为 `passed`，运行执行为 `blocked_external_runtime_evidence`。
